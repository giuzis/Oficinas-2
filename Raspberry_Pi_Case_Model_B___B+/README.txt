# Summary

This is a simple and modern case for single-board-computers like the Raspberry Pi Model 3 B+.There are multiple top's included, which have different configurations of openings. Additionally only four screws are needed to fix the SBC to the lower shell.

supported models (not guaranteed):

Raspberry Pi Model 3 B+
Raspberry Pi Model 3 B
Raspberry Pi Model 2 B+
Raspberry Pi Model 2 B
Raspberry Pi Model 1 B+

... and many more, with the same dimensions.

If you coincidentally want to use this case for a Libre Computer "Le Potato" [here](https://www.thingiverse.com/thing:3426091) is my special design for this SBC.
